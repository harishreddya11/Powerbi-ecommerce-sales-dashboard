# Power BI E-Commerce Sales Dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze E-Commerce sales performance. The dashboard provides key business insights into sales, profit, quantity sold, product performance, regional performance, and shipping trends.

## Key Features

### KPI Cards
- YTD Sales
- YTD Profit
- YTD Quantity Sold
- YTD Profit Margin

### Sales Analysis
- Sales by Category
- Year-over-Year (YoY) Comparison
- Sales Trend Analysis

### Product Performance
- Top 5 Products by YTD Sales
- Bottom 5 Products by YTD Sales

### Regional Insights
- Sales by Region
- Sales by State (Interactive Map)

### Shipping Analysis
- Sales by Shipping Type

### Customer Segmentation
- Consumer
- Corporate
- Home Office

## Tools & Technologies

- Power BI Desktop
- Power Query

- Data Visualization

## Business Insights

- Identified top-performing products contributing most to revenue.
- Compared sales performance across regions.
- Analyzed profit margins and sales trends.
- Evaluated shipping preferences of customers.
- Tracked year-to-date business performance using KPI metrics.
